Strata is a development framework that gives you a stable environment for creating Wordpress projects. It allows you to organize your code outside of the templating engine.

It keeps Wordpress's functions, filters and actions unmodified while doing so.

The development environment contains:

  • A stand-alone webserver
  • Generation and scaffolding of MVC files
  • Testing suite
  • Automated documentation tools
  • DB management tools

The framework allows:

  • PHP dependencies through Composer
  • Controller based page handling
  • Contained Model queries
  • ViewHelper tools
  • TDD approach
  • Same codebase for all website themes

Strata provides a development environment which features concepts often seen in other MVC solutions found in multiple programming languages. It eases the development of the more complex and tailored Wordpress applications by separating logic and presentation. In doing so, the code is easier to work on by teams of developers in the context of a business.
