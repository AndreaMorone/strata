Expressive application layer

2

Centralized SQL queries

Strata models reference Wordpress Custom Post Types and Taxonomies in a layer of abstraction that allows descriptive SQL query chaining.

Centralized query management eases development by simplifying complex fetches and by allowing the integration of new features easily.

3

Code with intent

Spend less time removing default Wordpress behavior through filters and spend more time in writing descriptive and testable application behavior.

With object oriented MVC programming and a set of development tools Strata will allow you to build an application without counting exclusively on plugins that 'almost work right' for your use case.

Get to know Strata

Strata provides a development environment which features concepts often seen in other MVC solutions found in multiple programming languages.

It eases the development of the more complex and tailored Wordpress applications by separating logic and presentation.

In doing so, the code is easier to work on by teams of developers in the context of a business.
