Strata is a development framework that gives you a stable environment for creating Wordpress projects. It allows you to organize your code outside of the templating engine.

It vows to keep Wordpress unmodified while doing so.

The development environment contains:

  • A stand-alone webserver
  • Generators of MVC files
  • Testing suite
  • Automated documentation suite
  • DB migration tools

The framework allows:

  • PHP dependencies through Composer
  • Controller based page handling
  • Contained Model queries
  • ViewHelper tools
  • TDD approach
  • Same codebase for all website themes

Strata provides a development environment which features concepts often seen in other MVC solutions of multiple languages. It eases the development of more complex and customized Wordpress applications by separating logic and presentation. In doing so, the code is easier to work on to teams of developers in the context of a business.
