Strata lays on top of Bedrock to give you a stable development environment for Wordpress that allows you to organize your code outside of the templating engine.

It vows to keep Wordpress unmodified while doing so.

The development environment contains:

  • A stand-alone webserver
  • Generators of MVC files
  • Testing suite
  • Documentation suite
  • DB migration tools

The framework allows:

  • PHP dependencies through Composer
  • Controller based page handling
  • Contained Model queries
  • ViewHelper tools
  • TDD approach
  • Same codebase for all website themes

Strata provides a development environment which features concepts often seen in other MVC solutions of multiple languages. It eases the development of more complex and custom Wordpress applications by separating logic and presentation. In doing so, the code is easier to work on for a team of developers in the context of a business.
