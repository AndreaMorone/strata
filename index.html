Wordpress MVC gives you a stable development environment that allows you to organize your code outside of the templating engine.

It vows to keep Wordpress unmodified while doing so.

The development environment contains:

  • A stand-alone webserver
  • Generators of MVC files
  • Testing suite
  • Documentation suite
  • Migration tools

The framework allows:

  • PHP dependencies through Composer
  • Controller page state handling
  • Contained Model queries
  • ViewHelper tools
  • Performance boost (unused code is not interpreted)
  • Code availability for all website themes

Wordpress MCV provides a development environment featuring concepts often seen in other MVC solutions of all languages. It eases the development of more complex and custom Wordpress applications by separating logic and presentation. In doing so, the code is easier to work on for a team of developers in the context of a business.
