Strata\Utility\Hash::sort

Sorts an array by any value, determined by a Set-compatible path

### Sort directions - `asc` Sort ascending. - `desc` Sort descending. ## Sort types - `regular` For regular sorting (don't change types) - `numeric` Compare values numerically - `string` Compare values as strings - `natural` Compare items as strings using "natural ordering" in a human friendly way. Will sort foo10 below foo2 as an example. Requires PHP 5.4 or greater or it will fallback to 'regular'
